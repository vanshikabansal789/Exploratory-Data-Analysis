# Food Delivery EDA | MySQL

## Project Summary
Exploratory Data Analysis on a Zomato-style food delivery 
dataset using MySQL. Analysed 1,500 orders across 5 tables 
to uncover revenue trends, customer behaviour, and delivery 
performance.

## Tables
- customers (300 rows)
- restaurants (20 rows)
- orders (1,500 rows)
- order_items (3,823 rows)
- delivery (903 rows)

## Key Business Questions Answered
- Which city drives the most revenue?
- Top 5 restaurants by revenue
- Month on month revenue growth
- March 2023 degrowth investigation
- Top 10 customers by spend
- Repeat vs one-time customer segmentation

## Skills Used
MySQL · JOINs · CTEs · Window Functions · 
LAG() · RANK() · ROW_NUMBER() · CASE WHEN
